# 49. 字母异位词分组

## 中等

## 相关标签
- 字母异位词

## 问题描述
给你一个字符串数组，请你将字母异位词组合在一起。可以按任意顺序返回结果列表。

字母异位词是由重新排列源单词的所有字母得到的一个新单词。

## 示例

### 示例 1
**输入:**
```plaintext
strs = ["eat", "tea", "tan", "ate", "nat", "bat"]
```
**输出:**
```plaintext
[["bat"], ["nat", "tan"], ["ate", "eat", "tea"]]
```

### 示例 2
**输入:**
```plaintext
strs = [""]
```
**输出:**
```plaintext
[[""]]
```

### 示例 3
**输入:**
```plaintext
strs = ["a"]
```
**输出:**
```plaintext
[["a"]]
```

## 提示
- 1 <= strs.length <= 10^4
- 0 <= strs[i].length <= 100
- strs[i] 仅包含小写字母
